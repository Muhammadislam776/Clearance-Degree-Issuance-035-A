# Clearance-Degree-Issuance-035-A

Initial UI step for the **Smart Student Clearance & Degree Issuance System**.

## Implemented in this step
- Attractive and professional **Signup** and **Login** page.
- Role options: **Student, Admin, Department Staff, Examiner**.
- Student-specific fields: name, roll number, session, department, email, password.
- Role-aware signup form with dynamic fields.
- Login form with role selection and same-password flow placeholder.
- Clean SaaS-style responsive design (blue/white/gray palette).

## Files
- `index.html`
- `styles.css`
- `script.js`

## Run locally
Open `index.html` in your browser.
